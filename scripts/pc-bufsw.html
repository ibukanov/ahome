pc-bufsw buffer switch for Emacs

A fast way to do a buffer switch in Emacs is to use pc-bufsw.el utility. I initially created it in an attempt to model a Ctrl-Tab like switch that you can find in Windows applications.

To use it make sure that emacs lisp require call will find it and put the following lines to your .emacs file: 

(require 'pc-bufsw)
(pc-bufsw::bind-keys [C-tab] [C-S-tab])
Or, if you use XEmacs, put it like (as suggested by Friedrich Dominicus, ) 
(require 'pc-bufsw)
(pc-bufsw::bind-keys [(control tab)] [ (control shift tab) ])

It may be more convenient to bind buffer switching to single keys. In fact, in my own .emacs file I have: 

(pc-bufsw::bind-keys [C-tab] [C-S-tab])
(pc-bufsw::bind-keys [f12] [f11])
In this way pressing single F12 is enough to switch to the previous buffer.
